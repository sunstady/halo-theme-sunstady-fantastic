## 使用方法

1. 克隆或者下载
2. 压缩为 zip 压缩包之后在后台的主题设置直接上传即可使用
3. 用 Halo 自带的程序拉取 `https://github.com/sunstady/halo-theme-sunstady-fantastic.git`
4. 当前版本 `1.5.0`，修改日志详见 `CHANGELOG.md`
